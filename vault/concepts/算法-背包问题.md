# 背包问题

背包问题是组合优化的经典 DP 问题：给定容量为 W 的背包和 n 个物品（各有重量和价值），在不超过容量的前提下最大化总价值。

## 0-1 背包

每件物品**选或不选**，只能选一次。

```python
def knapsack_01(weights, values, W):
    n = len(weights)
    # dp[i][w] = 前 i 件物品，容量 w 时的最大价值
    dp = [0] * (W + 1)
    
    for i in range(n):
        for w in range(W, weights[i] - 1, -1):  # 逆序，避免重复选
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    
    return dp[W]
# 时间 O(nW)，空间 O(W)
```

> [!important] 0-1 背包必须逆序遍历
> 正序遍历会导致同一物品被选多次（变成完全背包）。

## 完全背包

每件物品可选**无限次**。

```python
def knapsack_complete(weights, values, W):
    n = len(weights)
    dp = [0] * (W + 1)
    
    for i in range(n):
        for w in range(weights[i], W + 1):  # 正序，允许重复选
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i])
    
    return dp[W]
# 时间 O(nW)，空间 O(W)
```

## 多重背包

每件物品有**数量限制** `cnt[i]`。

```python
def knapsack_bounded(weights, values, counts, W):
    dp = [0] * (W + 1)
    n = len(weights)
    
    for i in range(n):
        # 二进制拆分优化
        k = 1
        remaining = counts[i]
        while remaining > 0:
            take = min(k, remaining)
            tw = take * weights[i]
            tv = take * values[i]
            for w in range(W, tw - 1, -1):
                dp[w] = max(dp[w], dp[w - tw] + tv)
            remaining -= take
            k <<= 1
    
    return dp[W]
# 二进制拆分后时间 O(nW log(max_cnt))
```

## 空间优化技巧

> [!tip] 滚动数组
> - **0-1 背包**：逆序遍历 `W → weight[i]`，一维数组即可
> - **完全背包**：正序遍历 `weight[i] → W`
> - **二维 → 一维**：空间从 O(nW) 降至 O(W)

> [!info] 复杂度分析
> - **时间**：O(nW) — 伪多项式时间（W 为数值，非输入位数）
> - **空间**：O(W)（优化后）/ O(nW)（原始）
