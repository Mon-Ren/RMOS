# 单调栈与单调队列

通过维护栈/队列的单调性，实现 O(n) 的高效区间查询。

## 单调栈

维护栈内元素单调递增或递减，用于找**下一个更大/更小元素**。

### 下一个更大元素（LeetCode 496）

```python
def next_greater(nums):
    """每个元素右边第一个比它大的元素"""
    n = len(nums)
    result = [-1] * n
    stack = []  # 存下标，栈底到栈顶递减
    
    for i in range(n):
        while stack and nums[i] > nums[stack[-1]]:
            idx = stack.pop()
            result[idx] = nums[i]
        stack.append(i)
    
    return result
# 时间 O(n)，空间 O(n)
```

### 柱状图最大矩形（LeetCode 84）

```python
def largest_rectangle(heights):
    stack = []  # 单调递增栈
    max_area = 0
    heights.append(0)  # 哨兵
    
    for i, h in enumerate(heights):
        while stack and heights[stack[-1]] > h:
            height = heights[stack.pop()]
            width = i if not stack else i - stack[-1] - 1
            max_area = max(max_area, height * width)
        stack.append(i)
    
    return max_area
```

### 接雨水（LeetCode 42）

```python
def trap(height):
    stack = []
    water = 0
    
    for i, h in enumerate(height):
        while stack and height[stack[-1]] < h:
            bottom = height[stack.pop()]
            if not stack:
                break
            width = i - stack[-1] - 1
            bounded = min(h, height[stack[-1]]) - bottom
            water += width * bounded
        stack.append(i)
    
    return water
```

## 单调队列

维护队列内元素单调性，用于求**滑动窗口内最值**。

### 滑动窗口最大值（LeetCode 239）

```python
from collections import deque

def max_sliding_window(nums, k):
    dq = deque()  # 存下标，队头到队尾递减
    result = []
    
    for i, num in enumerate(nums):
        # 移除超出窗口的元素
        while dq and dq[0] < i - k + 1:
            dq.popleft()
        # 维护单调性：移除所有小于当前值的
        while dq and nums[dq[-1]] < num:
            dq.pop()
        dq.append(i)
        
        if i >= k - 1:
            result.append(nums[dq[0]])
    
    return result
# 时间 O(n)，空间 O(k)
```

> [!tip] 单调栈模板
> ```python
> stack = []
> for i in range(n):
>     while stack and condition(nums[i], nums[stack[-1]]):
>         stack.pop()
>         # 处理逻辑
>     stack.append(i)
> ```

> [!tip] 单调队列模板
> ```python
> dq = deque()
> for i in range(n):
>     while dq and dq[0] < i - k + 1:  # 移除过期元素
>         dq.popleft()
>     while dq and condition(nums[i], nums[dq[-1]]):  # 维护单调性
>         dq.pop()
>     dq.append(i)
> ```

> [!info] 复杂度分析
> - **单调栈**：O(n) — 每个元素最多入栈出栈各一次
> - **单调队列**：O(n) — 同理
