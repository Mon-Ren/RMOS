# 前缀和

前缀和是一种预处理技术，通过对数组预计算累积和，将区间求和从 O(n) 降至 O(1)。

## 一维前缀和

```python
class PrefixSum:
    def __init__(self, arr):
        n = len(arr)
        self.prefix = [0] * (n + 1)
        for i in range(n):
            self.prefix[i + 1] = self.prefix[i] + arr[i]
    
    def query(self, left, right):
        """返回 arr[left..right] 的和（闭区间）"""
        return self.prefix[right + 1] - self.prefix[left]
# 建立 O(n)，查询 O(1)
```

## 二维前缀和（子矩阵求和）

```python
class PrefixSum2D:
    def __init__(self, matrix):
        m, n = len(matrix), len(matrix[0])
        self.prefix = [[0] * (n + 1) for _ in range(m + 1)]
        for i in range(1, m + 1):
            for j in range(1, n + 1):
                self.prefix[i][j] = (matrix[i-1][j-1]
                    + self.prefix[i-1][j]
                    + self.prefix[i][j-1]
                    - self.prefix[i-1][j-1])
    
    def query(self, r1, c1, r2, c2):
        """返回子矩阵 (r1,c1) 到 (r2,c2) 的和"""
        return (self.prefix[r2+1][c2+1]
              - self.prefix[r1][c2+1]
              - self.prefix[r2+1][c1]
              + self.prefix[r1][c1])
# 建立 O(mn)，查询 O(1)
```

## 差分数组

差分是前缀和的逆操作，用于**区间修改**：

```python
class DifferenceArray:
    def __init__(self, arr):
        self.n = len(arr)
        self.diff = [0] * (self.n + 1)
        self.diff[0] = arr[0]
        for i in range(1, self.n):
            self.diff[i] = arr[i] - arr[i - 1]
    
    def update(self, left, right, val):
        """将 arr[left..right] 每个元素加 val"""
        self.diff[left] += val
        self.diff[right + 1] -= val
    
    def result(self):
        """还原最终数组"""
        arr = [0] * self.n
        arr[0] = self.diff[0]
        for i in range(1, self.n):
            arr[i] = arr[i-1] + self.diff[i]
        return arr
# 区间修改 O(1)，还原 O(n)
```

### 前缀和 + 哈希表（子数组和）

```python
def subarray_sum(nums, k):
    """和为 k 的子数组个数（LeetCode 560）"""
    from collections import defaultdict
    count = 0
    prefix_sum = 0
    prefix_count = defaultdict(int)
    prefix_count[0] = 1
    
    for num in nums:
        prefix_sum += num
        count += prefix_count[prefix_sum - k]
        prefix_count[prefix_sum] += 1
    
    return count
```

> [!tip] 复杂度分析
> - **建立**：O(n) 一维 / O(mn) 二维
> - **查询**：O(1)
> - **差分区间修改**：O(1)
> - **空间**：O(n) / O(mn)

> [!info] 应用场景
> - 频繁查询区间和（在线查询）
> - 区间修改 + 最终还原（离线处理）
> - 孞矩阵求和（图像处理、动态规划）
> - LeetCode 303/304/560
