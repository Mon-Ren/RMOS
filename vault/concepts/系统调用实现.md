# 系统调用实现

xv6 系统调用的完整实现路径：从用户态 `INT` 指令到内核处理函数，再到返回用户态。

## 核心机制

### 用户态发起

用户程序通过 C 库函数发起系统调用，最终执行 `INT T_SYSCALL`（中断号 64）：

```asm
; usys.S — 自动生成的系统调用桩代码
.globl sys_fork
sys_fork:
  movl $SYS_fork, %eax    # 系统调用号放入 eax
  int $T_SYSCALL           # 触发软中断
  ret
```

系统调用号定义在 `syscall.h`：
```c
#define SYS_fork    1
#define SYS_exit    2
#define SYS_wait    3
#define SYS_pipe    4
#define SYS_read    5
// ...共 21 个系统调用
```

### 内核入口 — 陷阱处理

`INT` 指令触发 CPU 自动：
1. 切换到内核态（切换 TSS 中的 %esp）
2. 将用户态寄存器压入内核栈（trapframe）
3. 跳转到 IDT[64] 对应的处理函数

```c
// trap.c:20 — trap 分发
case T_SYSCALL:
  if(myproc()->killed)
    exit();
  myproc()->tf = tf;
  syscall();    // 进入系统调用分发
  if(myproc()->killed)
    exit();
  break;
```

### 参数获取

`syscall.c` 提供参数提取函数，从用户栈上读取参数：

```c
// syscall.c:15 — 读取第 n 个 int 参数
int argint(int n, int *ip) {
  return fetchint((myproc()->tf->esp) + 4 + 4*n, ip);
}

// syscall.c:34 — 读取指针参数（带边界检查）
int argptr(int n, char **pp, int size) {
  int i;
  if(argint(n, &i) < 0)
    return -1;
  if((uint)i >= curproc->sz || (uint)i+size > curproc->sz)
    return -1;
  *pp = (char*)i;
  return 0;
}
```

参数位置：`tf->esp + 4 + 4*n`
- `+4` 跳过返回地址
- `+4*n` 定位第 n 个参数

### 系统调用分发表

```c
// syscall.c:105 — 分发表
static int (*syscalls[])(void) = {
[SYS_fork]    sys_fork,
[SYS_exit]    sys_exit,
[SYS_wait]    sys_wait,
[SYS_pipe]    sys_pipe,
[SYS_read]    sys_read,
[SYS_write]   sys_write,
[SYS_close]   sys_close,
// ...
};

void syscall(void) {
  int num;
  struct proc *curproc = myproc();
  num = curproc->tf->eax;        // 系统调用号在 eax 中
  if(num > 0 && num < NELEM(syscalls) && syscalls[num]) {
    curproc->tf->eax = syscalls[num]();  // 返回值放回 eax
  } else {
    cprintf("%d %s: unknown sys call %d\n",
            curproc->pid, curproc->name, num);
    curproc->tf->eax = -1;
  }
}
```

### 返回用户态

系统调用函数执行完毕后，返回值写入 `tf->eax`，`trap()` 返回时执行 `IRET` 指令，CPU 自动从内核栈恢复用户态寄存器并切换回用户态。

## 完整流程

```
用户态 fork()
  → movl $SYS_fork, %eax
  → int $T_SYSCALL
    → CPU: 切换内核栈，压入 trapframe
    → IDT[64] → alltraps → trap()
    → trap(): case T_SYSCALL → syscall()
    → syscall(): num = tf->eax = 1
    → syscalls[1]() = sys_fork()
    → sys_fork(): 创建子进程，返回 pid
    → tf->eax = pid（返回值）
    → trap() 返回 → trapret
  → IRET → 用户态，eax = pid
```

## 关键要点
> xv6 系统调用通过 INT 指令触发，CPU 自动切换到内核态并保存用户寄存器到 trapframe。内核通过 eax 中的系统调用号查表分发，参数从用户栈读取，返回值写回 eax。
