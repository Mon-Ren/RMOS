# 二叉搜索树

BST 是满足"左 < 根 < 右"性质的二叉树，支持 O(h) 的查找、插入、删除。

## BST 性质

对于任意节点 `node`：
- `node.left` 及其所有后代 < `node.val`
- `node.right` 及其所有后代 > `node.val`
- **中序遍历得到严格递增序列**

## 查找

```python
def search(root, target):
    if not root:
        return None
    if target == root.val:
        return root
    elif target < root.val:
        return search(root.left, target)
    else:
        return search(root.right, target)
```

**时间**：O(h)，平衡时 O(log n)

## 插入

```python
def insert(root, val):
    if not root:
        return TreeNode(val)
    if val < root.val:
        root.left = insert(root.left, val)
    elif val > root.val:
        root.right = insert(root.right, val)
    # val == root.val 时不插入（或更新）
    return root
```

## 删除（最复杂）

```python
def delete(root, key):
    if not root:
        return None
    if key < root.val:
        root.left = delete(root.left, key)
    elif key > root.val:
        root.right = delete(root.right, key)
    else:
        # 找到要删除的节点
        if not root.left:
            return root.right       # 只有右子树
        if not root.right:
            return root.left        # 只有左子树
        # 左右子树都有：找后继（右子树最小值）
        succ = root.right
        while succ.left:
            succ = succ.left
        root.val = succ.val
        root.right = delete(root.right, succ.val)
    return root
```

## 有序性验证

```python
def is_valid_bst(root, lo=float('-inf'), hi=float('inf')):
    if not root:
        return True
    if root.val <= lo or root.val >= hi:
        return False
    return (is_valid_bst(root.left, lo, root.val)
            and is_valid_bst(root.right, root.val, hi))
```

## 复杂度

| 操作 | 平均（平衡） | 最坏（退化为链表） |
|------|-------------|-------------------|
| 查找 | O(log n) | O(n) |
| 插入 | O(log n) | O(n) |
| 删除 | O(log n) | O(n) |

**空间**：O(h)，存储 n 个节点需 O(n) 空间

## 退化问题

当按有序序列插入时，BST 退化为链表：

```
insert 1,2,3,4,5 → 全部右子树
    1
     \
      2
       \
        3
         \
          4
           \
            5
```

**解决方案**：自平衡 BST → [[AVL 树]]、[[红黑树]]

## 关键要点

> [!important] 中序遍历 = 排序
> BST 最有用的性质之一：中序遍历直接得到有序序列

> [!important] 删除有三种情况
> 叶子直接删、一个子树替换、两个子树找后继

> [!important] 普通 BST 不保证平衡
> 有序插入导致退化为 O(n)，需要自平衡变体
