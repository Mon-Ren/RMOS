# x86 寄存器概述

x86 架构提供多类寄存器，按功能分为通用、段、控制、调试等类别。理解寄存器是学习汇编的第一步。

## 寄存器分类

| 类别 | 寄存器 | 用途 |
|------|--------|------|
| 通用 | EAX, EBX, ECX, EDX, ESI, EDI, EBP, ESP | 数据运算、地址计算 |
| 段 | CS, DS, ES, FS, GS, SS | 内存分段（现代已弱化） |
| 指令指针 | EIP | 下一条指令地址 |
| 标志 | EFLAGS | 运算结果状态、控制位 |
| 控制 | CR0-CR4 | 分页、保护模式控制 |
| 调试 | DR0-DR7 | 硬件断点 |
| x87 FPU | ST(0)-ST(7) | 浮点运算 |
| MMX | MM0-MM7 | 64位 SIMD（复用 FPU） |
| SSE | XMM0-XMM7 | 128位 SIMD |
| AVX | YMM0-YMM7 | 256位 SIMD |

## 64 位扩展

x86-64 增加：
- 通用寄存器扩展到 64 位：RAX, RBX, RCX, RDX, RSI, RDI, RBP, RSP
- 新增 8 个通用寄存器：R8-R15
- SSE 扩展到 128 位 XMM → AVX 256 位 YMM

## 关键要点

> 寄存器是 CPU 内部最快的存储。汇编编程本质上就是操作这些寄存器和内存。每类寄存器有特定用途，但通用寄存器在多数指令中可以互换使用。
