# 位运算技巧

位运算直接操作二进制位，速度极快，在算法竞赛和系统编程中有大量巧妙应用。

## 基本运算符

| 运算符 | 含义 | 示例 |
|--------|------|------|
| `&` | 按位与 | `5 & 3 = 1` |
| `\|` | 按位或 | `5 \| 3 = 7` |
| `^` | 按位异或 | `5 ^ 3 = 6` |
| `~` | 按位取反 | `~5 = -6` |
| `<<` | 左移 | `5 << 1 = 10` |
| `>>` | 右移 | `5 >> 1 = 2` |

## 经典技巧

### n & (n-1) — 清除最低位的 1

```python
def is_power_of_two(n):
    return n > 0 and (n & (n - 1)) == 0

def count_ones(n):
    """计算二进制中 1 的个数"""
    count = 0
    while n:
        n &= n - 1
        count += 1
    return count
# 更简洁：bin(n).count('1')
```

### n & -n — 提取最低位的 1

```python
def lowest_bit(n):
    """返回最低位 1 对应的值，如 12(1100) -> 4(0100)"""
    return n & (-n)
```

### 异或技巧

```python
def swap(a, b):
    """不用临时变量交换"""
    a ^= b
    b ^= a
    a ^= b
    return a, b

def single_number(nums):
    """数组中只有一个数出现一次，其余出现两次"""
    result = 0
    for n in nums:
        result ^= n
    return result
# 异或性质：a ^ a = 0, a ^ 0 = a
```

### 位掩码枚举子集

```python
def enumerate_subsets(nums):
    """枚举所有子集"""
    n = len(nums)
    result = []
    for mask in range(1 << n):  # 0 到 2^n - 1
        subset = []
        for i in range(n):
            if mask & (1 << i):
                subset.append(nums[i])
        result.append(subset)
    return result

def enumerate_subsets_fast(nums):
    """高效子集枚举"""
    n = len(nums)
    for mask in range(1 << n):
        subset = []
        s = mask
        while s:
            i = (s & -s).bit_length() - 1  # 最低位索引
            subset.append(nums[i])
            s &= s - 1
        yield subset
```

### 常用位操作速查

```python
x | (1 << i)     # 将第 i 位设为 1
x & ~(1 << i)    # 将第 i 位设为 0
x ^ (1 << i)     # 翻转第 i 位
x & (1 << i)     # 检查第 i 位是否为 1
(x >> i) & 1     # 取第 i 位的值
```

> [!tip] 复杂度分析
> - 位运算本身 O(1)
> - 枚举子集 O(2ⁿ · n)
> - 位运算比算术运算快得多（硬件级别）

> [!info] 竞赛应用
> - 状态压缩 DP（旅行商问题、集合覆盖）
> - 快速判断奇偶、幂次
> - 异或哈希（字符串 / 树的判等）
