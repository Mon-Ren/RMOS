# 二叉树

二叉树是每个节点最多有两个子节点的树结构，是面试和工程中最常见的树类型。

## 节点定义

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
```

## 四种遍历

### 递归实现

```python
# 前序：根 → 左 → 右
def preorder(root):
    if not root:
        return []
    return [root.val] + preorder(root.left) + preorder(root.right)

# 中序：左 → 根 → 右
def inorder(root):
    if not root:
        return []
    return inorder(root.left) + [root.val] + inorder(root.right)

# 后序：左 → 右 → 根
def postorder(root):
    if not root:
        return []
    return postorder(root.left) + postorder(root.right) + [root.val]

# 层序：BFS
from collections import deque
def level_order(root):
    if not root:
        return []
    result, queue = [], deque([root])
    while queue:
        level = []
        for _ in range(len(queue)):
            node = queue.popleft()
            level.append(node.val)
            if node.left: queue.append(node.left)
            if node.right: queue.append(node.right)
        result.append(level)
    return result
```

### 迭代实现（用栈）

```python
# 前序迭代
def preorder_iterative(root):
    if not root:
        return []
    result, stack = [], [root]
    while stack:
        node = stack.pop()
        result.append(node.val)
        if node.right: stack.append(node.right)  # 右先入后出
        if node.left: stack.append(node.left)
    return result

# 中序迭代
def inorder_iterative(root):
    result, stack = [], []
    curr = root
    while curr or stack:
        while curr:
            stack.append(curr)
            curr = curr.left
        curr = stack.pop()
        result.append(curr.val)
        curr = curr.right
    return result
```

## 深度、高度、直径

```python
# 最大深度（高度）
def max_depth(root):
    if not root:
        return 0
    return 1 + max(max_depth(root.left), max_depth(root.right))

# 直径：任意两节点间最长路径
def diameter(root):
    best = [0]
    def depth(node):
        if not node:
            return 0
        L = depth(node.left)
        R = depth(node.right)
        best[0] = max(best[0], L + R)  # 经过当前节点的路径
        return 1 + max(L, R)
    depth(root)
    return best[0]
```

## 镜像与对称

```python
# 镜像（翻转）
def mirror(root):
    if not root:
        return None
    root.left, root.right = mirror(root.right), mirror(root.left)
    return root

# 判断对称
def is_symmetric(root):
    def is_mirror(t1, t2):
        if not t1 and not t2:
            return True
        if not t1 or not t2:
            return False
        return (t1.val == t2.val
                and is_mirror(t1.left, t2.right)
                and is_mirror(t1.right, t2.left))
    return is_mirror(root, root)
```

## 复杂度

| 操作 | 时间 | 空间 |
|------|------|------|
| 遍历 | O(n) | O(h) 栈空间，h = 树高 |
| 查找 | O(n) | O(h) |
| 插入 | O(h) | O(h) |

> h = log n（平衡时），h = n（退化时）

## 关键要点

> [!important] 前序 = 根在前，中序 = 根在中，后序 = 根在后
> 中序遍历 BST 得到有序序列

> [!important] 递归是最自然的树操作方式
> 迭代实现本质是用栈模拟递归调用栈

> [!important] 树的问题多数可分解为子树问题
> 典型模式：定义递归函数处理子树，合并结果
