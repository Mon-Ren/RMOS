# 零初始化与值初始化

C++ 有三种初始化方式——理解它们的差异是避免未初始化变量的关键。

## 三种初始化

```cpp
// 1. 默认初始化：T var;
int x;              // 未初始化（内置类型）——值不确定
Widget w;           // 调用默认构造函数

// 2. 值初始化：T var() 或 T var{}；
int y = int();      // 零初始化 → y = 0
int z{};            // 零初始化 → z = 0
Widget w2{};        // 调用默认构造函数

// 3. 零初始化：
// 静态/线程存储期的变量自动零初始化
static int s;       // s = 0
int g;              // 全局变量 → g = 0
// 局部变量默认不零初始化
void foo() { int x; }  // x 未初始化！
```

## 统一初始化（C++11）

```cpp
// 花括号初始化：值初始化或聚合初始化
int a{};            // 0
int b{42};          // 42
std::string s{};    // 空字符串
std::vector<int> v{1, 2, 3};  // 聚合初始化

// 防止窄化转换
int x = 3.14;       // OK：截断为 3（有警告）
// int y{3.14};      // 编译错误：防止窄化

// 最令人头疼的初始化
Widget w1();        // 这是函数声明！不是对象定义
Widget w2{};        // 这是对象定义
```

## 关键要点

> `{}` 初始化是最佳实践——值初始化（内置类型为 0）、防止窄化、消除"最令人头疼的解析"。

> 局部变量默认不初始化——用 `int x{};` 或 `int x = 0;` 保证初始化。
