# 回溯法

回溯法通过系统地搜索所有可能的解（构建搜索树），并在发现当前路径不可能得到解时剪枝返回，从而高效地解决问题。

## 核心模板

```python
def backtrack(path, choices):
    if is_solution(path):
        result.append(path[:])
        return
    
    for choice in choices:
        if not is_valid(choice):  # 剪枝
            continue
        path.append(choice)       # 选择
        backtrack(path, choices)  # 递归
        path.pop()                # 撤销
```

## 经典问题

### 全排列

```python
def permute(nums):
    result = []
    
    def backtrack(path, remaining):
        if not remaining:
            result.append(path[:])
            return
        for i in range(len(remaining)):
            path.append(remaining[i])
            backtrack(path, remaining[:i] + remaining[i+1:])
            path.pop()
    
    backtrack([], nums)
    return result
# 时间 O(n! · n)，空间 O(n)
```

### N 皇后

```python
def solve_n_queens(n):
    result = []
    
    def backtrack(row, cols, diag1, diag2, board):
        if row == n:
            result.append(["".join(r) for r in board])
            return
        for col in range(n):
            if col in cols or (row-col) in diag1 or (row+col) in diag2:
                continue  # 剪枝
            board[row][col] = 'Q'
            backtrack(row+1, cols|{col}, diag1|{row-col},
                      diag2|{row+col}, board)
            board[row][col] = '.'
    
    board = [['.' for _ in range(n)] for _ in range(n)]
    backtrack(0, set(), set(), set(), board)
    return result
```

### 数独求解

```python
def solve_sudoku(board):
    rows = [set() for _ in range(9)]
    cols = [set() for _ in range(9)]
    boxes = [set() for _ in range(9)]
    empty = []
    
    for i in range(9):
        for j in range(9):
            if board[i][j] != '.':
                val = board[i][j]
                rows[i].add(val)
                cols[j].add(val)
                boxes[i//3*3+j//3].add(val)
            else:
                empty.append((i, j))
    
    def backtrack(idx):
        if idx == len(empty):
            return True
        i, j = empty[idx]
        b = i//3*3+j//3
        for c in '123456789':
            if c not in rows[i] and c not in cols[j] and c not in boxes[b]:
                board[i][j] = c
                rows[i].add(c); cols[j].add(c); boxes[b].add(c)
                if backtrack(idx+1):
                    return True
                board[i][j] = '.'
                rows[i].discard(c); cols[j].discard(c); boxes[b].discard(c)
        return False
    
    backtrack(0)
```

## 剪枝策略

> [!tip] 常见剪枝手段
> 1. **可行性剪枝**：当前选择不满足约束，直接跳过
> 2. **最优性剪枝**：当前方案不可能优于已知最优解，提前返回
> 3. **对称性剪枝**：避免搜索等价的对称状态
> 4. **排序剪枝**：先搜索"更可能成功"的分支

> [!info] 复杂度分析
> - 通常为指数级 O(2ⁿ) 或 O(n!)
> - 剪枝可将实际搜索空间大幅减少
> - 好的剪枝策略可能将不可解变为可解
