# 链表

链表是最基础的动态数据结构，节点通过指针串联，无需连续内存。

## 链表类型

### 单链表

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next
```

### 双向链表

```python
class DoublyListNode:
    def __init__(self, val=0, prev=None, next=None):
        self.val = val
        self.prev = prev
        self.next = next
```

### 循环链表
尾节点的 next 指回头节点。

## 经典操作

### 反转链表

```python
# 迭代 — O(n) 时间, O(1) 空间
def reverse(head):
    prev = None
    curr = head
    while curr:
        next_node = curr.next
        curr.next = prev
        prev = curr
        curr = next_node
    return prev

# 递归 — O(n) 时间, O(n) 栈空间
def reverse_recursive(head):
    if not head or not head.next:
        return head
    new_head = reverse_recursive(head.next)
    head.next.next = head
    head.next = None
    return new_head
```

### 快慢指针找中间节点

```python
def find_middle(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    return slow  # fast 到终点时 slow 在中间
```

### 检测环（Floyd 判圈）

```python
def has_cycle(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow == fast:
            return True
    return False
```

### 合并两个有序链表

```python
def merge_two_lists(l1, l2):
    dummy = ListNode(0)  # 哨兵节点
    curr = dummy
    while l1 and l2:
        if l1.val <= l2.val:
            curr.next = l1
            l1 = l1.next
        else:
            curr.next = l2
            l2 = l2.next
        curr = curr.next
    curr.next = l1 or l2
    return dummy.next
```

## 哨兵节点技巧

使用虚拟头节点 (dummy) 避免处理头节点变化的边界情况：

```python
# 不用哨兵：需要单独处理 head 删除
# 用哨兵：统一逻辑
dummy = ListNode(0)
dummy.next = head
# ... 统一的删除逻辑 ...
return dummy.next
```

## 链表 vs 数组

| 操作 | 数组 | 链表 |
|------|------|------|
| 随机访问 | O(1) | O(n) |
| 头部插入 | O(n) | O(1) |
| 尾部插入 | O(1) 均摊 | O(n) / O(1) 双指针 |
| 中间插入 | O(n) | O(1)（已有引用时） |
| 空间 | 连续 | 额外指针开销 |
| 缓存 | ✅ 友好 | ❌ 不友好 |

## 复杂度

- **访问**：O(n)
- **查找**：O(n)
- **插入/删除**：O(1)（已知前驱节点时）
- **空间**：O(n)，每个元素额外一个指针（双向则两个）

## 关键要点

> [!important] 快慢指针是链表的核心技巧
> 找中间、判环、找倒数第 k 个——都是快慢指针的变体

> [!important] 哨兵节点简化代码
> 任何涉及头节点可能改变的操作，都用 dummy node

> [!important] 链表操作注意断链顺序
> 反转、合并等操作中，先保存 next 再修改指针
