CMP 和 TEST 指令不保存计算结果，只更新标志寄存器，是条件跳转的基础——先比较，后跳转。

## CMP — 比较 (Compare)

CMP = SUB 不保存结果，只设 [[标志寄存器 EFLAGS]]。

```asm
cmp eax, ebx       ; 计算 EAX - EBX，结果丢弃，只设标志
cmp eax, 42        ; EAX 和 42 比较
cmp BYTE PTR [var], 0  ; 检查变量是否为 0
```

### CMP 设置的标志位

```
CMP A, B  等价于  A - B

结果为零：  ZF = 1  (A == B)
结果非零：  ZF = 0  (A != B)
产生借位：  CF = 1  (A < B，无符号)
无借位：    CF = 0  (A >= B，无符号)
有符号溢出：OF = 1
```

### CMP 结果与标志对应关系

```
CMP eax, ebx:

无符号比较：             有符号比较：
  CF=1 → eax < ebx       SF≠OF → eax < ebx
  CF=0 → eax >= ebx      SF=OF → eax >= ebx
  CF=0 ∧ ZF=0 → eax > ebx  SF=OF ∧ ZF=0 → eax > ebx
  ZF=1 → eax == ebx       ZF=1 → eax == ebx
```

```asm
; 无符号比较
cmp eax, ebx
ja  above          ; eax > ebx（无符号）
jae above_equal    ; eax >= ebx
jb  below          ; eax < ebx
jbe below_equal    ; eax <= ebx

; 有符号比较
cmp eax, ebx
jg  greater        ; eax > ebx（有符号）
jge greater_equal  ; eax >= ebx
jl  less           ; eax < ebx
jle less_equal     ; eax <= ebx
```

## TEST — 测试 (Test)

TEST = AND 不保存结果，只设标志。

```asm
test eax, ebx      ; 计算 EAX & EBX，结果丢弃，只设标志
test eax, eax      ; 检查 EAX 是否为 0（比 CMP eax, 0 短）
test al, 0x01      ; 检查最低位
```

### TEST 设置的标志位

```
TEST A, B  等价于  A & B

结果为零：  ZF = 1  (A & B == 0)
结果非零：  ZF = 0  (A & B != 0)
CF = 0（总是清零）
OF = 0（总是清零）
SF = 结果的最高位
PF = 结果低 8 位的奇偶性
```

```asm
; 检查 EAX 是否为 0
test eax, eax
jz  is_zero        ; ZF=1 → EAX == 0
jnz is_nonzero     ; ZF=0 → EAX != 0

; 检查特定位是否设置
test eax, 0x80     ; 检查第 7 位
jnz bit7_set       ; 位为 1 则跳转

; 检查奇偶
test eax, 1
jz  is_even        ; 最低位为 0 → 偶数
jnz is_odd         ; 最低位为 1 → 奇数

; 检查多个位是否同时为 0
test eax, 0x0F     ; 检查低 4 位
jz  low_nibble_zero  ; 低 4 位全为 0
```

## CMP vs TEST 的区别

```asm
; CMP 做减法，TEST 做与法

cmp eax, 0         ; EAX - 0：检查是否为零
test eax, eax      ; EAX & EAX：检查是否为零
; 两者效果相同，但 TEST 更短（2 字节 vs 5 字节）

; CMP 可以比较两个不同的值
cmp eax, 42        ; EAX == 42?
cmp eax, ebx       ; EAX == EBX?

; TEST 用于检测位模式
test eax, 0xFF     ; 低 8 位是否全为 0？
test eax, ebx      ; EAX 和 EBX 是否有共同位？
```

## 标志位变化详解

| 指令 | CF | ZF | SF | OF | PF | AF |
|------|----|----|----|----|----|----|
| CMP A,B (A==B) | 0 | 1 | 0 | 0 | 1 | — |
| CMP A,B (A<B unsigned) | 1 | 0 | — | — | — | — |
| CMP A,B (A<B signed) | — | 0 | — | — | — | — |
| TEST A,B (结果=0) | 0 | 1 | 0 | 0 | 1 | — |
| TEST A,B (结果≠0) | 0 | 0 | MSB | 0 | PF | — |

> [!tip] 关键要点
> - CMP = SUB 不存结果，TEST = AND 不存结果
> - CMP 后用 JA/JB/JAE/JBE（无符号）或 JG/JL/JGE/JLE（有符号）
> - TEST reg, reg 是检查零值的最优方式（比 CMP reg, 0 更短）
> - TEST 用于位检测，CMP 用于值比较
> - 理解 CF vs OF 的区别：CF 用于无符号判断，OF 用于有符号判断
