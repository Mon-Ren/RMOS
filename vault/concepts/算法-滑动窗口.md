# 滑动窗口

滑动窗口用两个指针维护一个"窗口"，通过右指针扩张和左指针收缩来高效求解子数组/子串问题。

## 两种类型

### 固定窗口

窗口大小固定为 k：

```python
def max_sum_subarray(arr, k):
    """大小为 k 的子数组最大和"""
    window_sum = sum(arr[:k])
    max_sum = window_sum
    for i in range(k, len(arr)):
        window_sum += arr[i] - arr[i - k]
        max_sum = max(max_sum, window_sum)
    return max_sum
```

### 变长窗口

```python
def min_window_template(s, condition):
    """变长窗口通用模板"""
    left = 0
    window_state = {}  # 窗口状态
    result = None
    
    for right in range(len(s)):
        # 1. 扩张窗口：将 s[right] 加入状态
        update_state(window_state, s[right])
        
        # 2. 收缩窗口：当窗口满足某种条件时
        while is_valid(window_state):
            # 更新结果（如取最小长度）
            result = update_result(result, left, right)
            # 移除 s[left]，左指针右移
            remove_from_state(window_state, s[left])
            left += 1
    
    return result
```

## 经典例题

### 最长无重复子串（LeetCode 3）

```python
def length_of_longest_substring(s):
    char_index = {}
    left = 0
    max_len = 0
    
    for right in range(len(s)):
        if s[right] in char_index and char_index[s[right]] >= left:
            left = char_index[s[right]] + 1
        char_index[s[right]] = right
        max_len = max(max_len, right - left + 1)
    
    return max_len
# 时间 O(n)，空间 O(min(n, m))，m 为字符集大小
```

### 最小覆盖子串（LeetCode 76）

```python
from collections import Counter

def min_window(s, t):
    need = Counter(t)
    missing = len(t)
    left = 0
    start, min_len = 0, float('inf')
    
    for right in range(len(s)):
        if need[s[right]] > 0:
            missing -= 1
        need[s[right]] -= 1
        
        while missing == 0:
            if right - left + 1 < min_len:
                min_len = right - left + 1
                start = left
            need[s[left]] += 1
            if need[s[left]] > 0:
                missing += 1
            left += 1
    
    return "" if min_len == float('inf') else s[start:start+min_len]
```

> [!tip] 复杂度分析
> - **时间**：O(n) — 左右指针各遍历数组一次
> - **空间**：O(k) — k 为字符集或状态大小

> [!info] 滑动窗口的本质
> 滑动窗口 = **双指针** + **单调性**。窗口内的某种性质（如"无重复"）具有单调性：扩大窗口可能破坏性质，缩小窗口可能恢复性质。
