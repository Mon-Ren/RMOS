# 概率与随机算法

随机算法利用随机性来简化设计、降低复杂度或在不确定环境下给出高效近似解。

## 两类随机算法

> [!info] 分类对比
> | 类型 | 特征 | 代表 |
> |------|------|------|
> | **蒙特卡罗**（Monte Carlo） | 结果可能错误，但运行时间确定 | Miller-Rabin 素性测试 |
> | **拉斯维加斯**（Las Vegas） | 结果一定正确，但运行时间随机 | 随机化快速排序 |

## 蓄水池抽样（Reservoir Sampling）

从数据流中等概率抽取 k 个样本，O(n) 时间，O(k) 空间。

```python
import random

def reservoir_sample(stream, k):
    """从数据流中等概率抽取 k 个"""
    reservoir = []
    
    for i, item in enumerate(stream):
        if i < k:
            reservoir.append(item)
        else:
            j = random.randint(0, i)
            if j < k:
                reservoir[j] = item
    
    return reservoir
# 每个元素被选中的概率 = k/n
```

## 随机化快速排序

随机选择 pivot 避免最坏情况：

```python
def randomized_quicksort(arr, lo=0, hi=None):
    if hi is None:
        hi = len(arr) - 1
    if lo >= hi:
        return
    
    # 随机选择 pivot
    pivot_idx = random.randint(lo, hi)
    arr[pivot_idx], arr[hi] = arr[hi], arr[pivot_idx]
    
    pivot = arr[hi]
    i = lo
    for j in range(lo, hi):
        if arr[j] < pivot:
            arr[i], arr[j] = arr[j], arr[i]
            i += 1
    arr[i], arr[hi] = arr[hi], arr[i]
    
    randomized_quicksort(arr, lo, i - 1)
    randomized_quicksort(arr, i + 1, hi)
# 期望 O(n log n)，最坏 O(n²) 但概率极低
```

## 布隆过滤器（Bloom Filter）

空间高效的概率数据结构，判断元素**是否可能存在**（可能误判存在，不会误判不存在）。

```python
import hashlib

class BloomFilter:
    def __init__(self, size, hash_count):
        self.size = size
        self.hash_count = hash_count
        self.bits = [False] * size
    
    def _hashes(self, item):
        result = []
        for i in range(self.hash_count):
            h = int(hashlib.md5(f"{item}{i}".encode()).hexdigest(), 16)
            result.append(h % self.size)
        return result
    
    def add(self, item):
        for h in self._hashes(item):
            self.bits[h] = True
    
    def might_contain(self, item):
        return all(self.bits[h] for h in self._hashes(item))
```

### 误判率

给定位数组大小 m、元素数量 n、哈希函数数 k：
- 最优 k = (m/n) · ln2
- 误判率 ≈ (1 - e^(-kn/m))^k

> [!tip] 复杂度分析
> - **蓄水池抽样**：O(n) 时间，O(k) 空间
> - **随机快排**：期望 O(n log n)
> - **布隆过滤器**：O(k) 插入/查询，k 为哈希函数数
> - **空间**：布隆过滤器仅需 ~10 bits/元素（1% 误判率）

> [!info] 哈希中的随机化
> - **随机哈希种子**：防 HashDoS 攻击
> - **一致性哈希**：分布式系统中均匀分配负载
> - **局部敏感哈希（LSH）**：近似最近邻搜索
