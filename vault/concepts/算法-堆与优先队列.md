# 堆与优先队列

堆是一种特殊的完全二叉树，支持高效获取最大/最小值。优先队列是基于堆实现的抽象数据类型。

## 二叉堆的性质

**最大堆**：父节点 ≥ 子节点
**最小堆**：父节点 ≤ 子节点

用**数组**表示完全二叉树：
- 父节点：`parent(i) = (i-1) // 2`
- 左子：`left(i) = 2*i + 1`
- 右子：`right(i) = 2*i + 2`

```
数组：[9, 7, 6, 5, 3, 1]

树结构：
        9 (0)
       /     \
     7 (1)   6 (2)
    /   \    /
  5(3) 3(4) 1(5)
```

## 核心操作

```python
class MinHeap:
    def __init__(self):
        self.data = []

    def push(self, val):                # O(log n)
        self.data.append(val)
        self._sift_up(len(self.data) - 1)

    def pop(self):                      # O(log n)
        if not self.data:
            raise IndexError
        self.data[0], self.data[-1] = self.data[-1], self.data[0]
        val = self.data.pop()
        self._sift_down(0)
        return val

    def peek(self):                     # O(1)
        return self.data[0] if self.data else None

    def _sift_up(self, i):
        while i > 0:
            parent = (i - 1) // 2
            if self.data[i] < self.data[parent]:
                self.data[i], self.data[parent] = self.data[parent], self.data[i]
                i = parent
            else:
                break

    def _sift_down(self, i):
        n = len(self.data)
        while True:
            smallest = i
            l, r = 2*i+1, 2*i+2
            if l < n and self.data[l] < self.data[smallest]:
                smallest = l
            if r < n and self.data[r] < self.data[smallest]:
                smallest = r
            if smallest != i:
                self.data[i], self.data[smallest] = self.data[smallest], self.data[i]
                i = smallest
            else:
                break
```

## Floyd 建堆算法

从最后一个非叶节点开始，逐个向下调整：

```python
def heapify(arr):
    """O(n) 时间建堆"""
    n = len(arr)
    for i in range(n // 2 - 1, -1, -1):
        _sift_down(arr, n, i)
```

**为什么是 O(n) 而不是 O(n log n)**：
- 高度 h 的节点数约 n/2^(h+1)
- 每个节点最多下调 h 步
- 总代价 = Σ(h=0 to log n) n/2^(h+1) · h < n

## 经典应用

### Top-K 问题

```python
import heapq

def top_k(nums, k):
    """找第 K 大元素 — O(n log k)"""
    heap = []
    for num in nums:
        heapq.heappush(heap, num)
        if len(heap) > k:
            heapq.heappop(heap)
    return heap[0]  # 堆顶是第 K 大
```

### 合并 K 个有序链表

```python
def merge_k_lists(lists):
    heap = []
    for i, node in enumerate(lists):
        if node:
            heapq.heappush(heap, (node.val, i, node))

    dummy = ListNode(0)
    curr = dummy
    while heap:
        val, i, node = heapq.heappop(heap)
        curr.next = node
        curr = curr.next
        if node.next:
            heapq.heappush(heap, (node.next.val, i, node.next))
    return dummy.next
```

### Dijkstra 最短路径

```python
import heapq

def dijkstra(graph, start):
    dist = {start: 0}
    heap = [(0, start)]
    while heap:
        d, u = heapq.heappop(heap)
        if d > dist.get(u, float('inf')):
            continue
        for v, w in graph[u]:
            nd = d + w
            if nd < dist.get(v, float('inf')):
                dist[v] = nd
                heapq.heappush(heap, (nd, v))
    return dist
```

## 复杂度

| 操作 | 时间 |
|------|------|
| push | O(log n) |
| pop | O(log n) |
| peek | O(1) |
| 建堆 | O(n) |
| 空间 | O(n) |

## 关键要点

> [!important] 堆 ≠ 排序的数组
> 堆只保证父子关系有序，兄弟之间无序

> [!important] Python heapq 是最小堆
> 取最大值可存负值或用自定义比较

> [!important] Top-K 用堆是经典面试题
> 维护大小为 K 的最小堆，O(n log k)
