# 栈与队列

栈 (LIFO) 和队列 (FIFO) 是最基础的线性数据结构，广泛用于算法设计。

## 栈 (Stack)

**后进先出 (LIFO)**：只在一端操作。

```python
class Stack:
    def __init__(self):
        self.items = []

    def push(self, x):      # O(1)
        self.items.append(x)

    def pop(self):           # O(1)
        return self.items.pop()

    def peek(self):          # O(1)
        return self.items[-1]

    def is_empty(self):
        return len(self.items) == 0
```

**应用场景**：
- 函数调用栈
- 括号匹配
- 表达式求值
- 单调栈（见下文）
- DFS

## 队列 (Queue)

**先进先出 (FIFO)**：一端入，另一端出。

```python
from collections import deque

class Queue:
    def __init__(self):
        self.items = deque()

    def enqueue(self, x):    # O(1)
        self.items.append(x)

    def dequeue(self):       # O(1)
        return self.items.popleft()

    def peek(self):
        return self.items[0]
```

**应用场景**：
- BFS 层序遍历
- 任务调度
- 消息队列
- 滑动窗口
- 单调队列

## 单调栈

维护栈内元素单调递增/递减，高效解决"下一个更大/更小元素"问题。

```python
# 下一个更大元素
def next_greater(nums):
    n = len(nums)
    result = [-1] * n
    stack = []  # 存下标，栈顶到栈底单调递减
    for i in range(n):
        while stack and nums[i] > nums[stack[-1]]:
            result[stack.pop()] = nums[i]
        stack.append(i)
    return result

# 示例：[2, 1, 2, 4, 3]
# 输出：[4, 2, 4, -1, -1]
```

**时间复杂度**：O(n)（每个元素最多入栈出栈各一次）

## 单调队列

维护队列内元素单调递增/递减，常用于滑动窗口最值。

```python
# 滑动窗口最大值
from collections import deque

def max_sliding_window(nums, k):
    dq = deque()  # 存下标，队首到队尾单调递减
    result = []
    for i, num in enumerate(nums):
        # 移除超出窗口的元素
        while dq and dq[0] < i - k + 1:
            dq.popleft()
        # 维护单调性
        while dq and nums[dq[-1]] < num:
            dq.pop()
        dq.append(i)
        if i >= k - 1:
            result.append(nums[dq[0]])
    return result
```

**时间复杂度**：O(n)

## 两个栈实现队列

```python
class QueueWithStacks:
    def __init__(self):
        self.s1 = []  # 入栈
        self.s2 = []  # 出栈

    def enqueue(self, x):       # O(1)
        self.s1.append(x)

    def dequeue(self):           # 均摊 O(1)
        if not self.s2:
            while self.s1:
                self.s2.append(self.s1.pop())
        return self.s2.pop()
```

> 两个队列也可实现栈，但不如此方案常用。

## 复杂度

| 操作 | 栈 | 队列 |
|------|----|------|
| 插入 | O(1) push | O(1) enqueue |
| 删除 | O(1) pop | O(1) dequeue |
| 查看 | O(1) peek | O(1) front |
| 空间 | O(n) | O(n) |

## 关键要点

> [!important] 单调栈解决"下一个更大元素"类问题
> 遍历一遍，O(n) 时间，面试高频

> [!important] 单调队列是滑动窗口最值的最优解
> 比堆方案 O(n log k) 更好，达到 O(n)

> [!important] Python 的 list 不适合当队列
> list.pop(0) 是 O(n)，用 collections.deque
