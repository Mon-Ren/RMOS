x86 架构提供 8 个 32 位通用寄存器，每个都有特定的历史用途和在现代编程中的约定用法。

## 8 个通用寄存器

### EAX — 累加器 (Accumulator)

历史上用于算术运算的主寄存器。许多指令隐式使用 EAX，如 `MUL`、`DIV`、`LOOP` 等。函数返回值通常也存放于此。

```asm
; 算术运算隐式使用 EAX
mov eax, 10
mul ebx          ; 结果 EDX:EAX = EAX * EBX

; 函数返回值
mov eax, 42      ; 返回 42
ret
```

### EBX — 基址寄存器 (Base)

常用于存放基地址，指向数据结构的起始位置。在 32 位 PIC（位置无关代码）中用作 GOT 基址。

```asm
; 作为数组基址
mov ebx, array_base
mov eax, [ebx + ecx*4]   ; 取 array[ecx]
```

### ECX — 计数器 (Counter)

循环计数的首选寄存器。`LOOP`、`REP` 前缀、`SHL/SHR` 等指令隐式使用 ECX。

```asm
; 循环计数
mov ecx, 100
loop_start:
    ; ... 循环体 ...
    dec ecx
    jnz loop_start

; 字符串复制
mov ecx, len
rep movsb
```

### EDX — 数据寄存器 (Data)

常与 EAX 配合用于乘除法的高位扩展。I/O 操作也常用 EDX 存放端口号。

```asm
; 除法：EDX:EAX / EBX
xor edx, edx      ; 清除高位
mov eax, 100
mov ebx, 7
div ebx            ; EAX=商(14), EDX=余数(2)
```

### ESI — 源索引 (Source Index)

字符串操作中指向源数据。`MOVSB`、`LODSB`、`CMPSB` 隐式使用。

```asm
lea esi, [source_string]
lea edi, [dest_string]
mov ecx, 20
rep movsb          ; 从 [ESI] 复制到 [EDI]
```

### EDI — 目标索引 (Destination Index)

字符串操作中指向目标数据。`MOVSB`、`STOSB`、`SCASB` 隐式使用。

```asm
lea edi, [buffer]
mov al, 0
mov ecx, 1024
rep stosb          ; 用 0 填充 1024 字节
```

### EBP — 基址指针 (Base Pointer)

栈帧的参考点。`[EBP+offset]` 用于访问局部变量和函数参数，是 [[栈帧结构]] 的核心。

```asm
; 函数序言
push ebp
mov ebp, esp
sub esp, 16        ; 分配 16 字节局部变量

; 访问参数和局部变量
mov eax, [ebp+8]   ; 第一个参数
mov ebx, [ebp-4]   ; 第一个局部变量
```

### ESP — 栈指针 (Stack Pointer)

始终指向栈顶。不应作为通用计算使用，否则会破坏栈。

```asm
push eax       ; ESP -= 4, [ESP] = EAX
pop ebx        ; EBX = [ESP], ESP += 4
```

## 8/16/8-High 别名

每个 32 位寄存器可以分段访问：

| 32位 | 16位 | 低8位 | 高8位(仅前4个) |
|------|------|-------|----------------|
| EAX  | AX   | AL    | AH             |
| EBX  | BX   | BL    | BH             |
| ECX  | CX   | CL    | CH             |
| EDX  | DX   | DL    | DH             |
| ESI  | SI   | —     | —              |
| EDI  | DI   | —     | —              |
| EBP  | BP   | —     | —              |
| ESP  | SP   | —     | —              |

```asm
mov eax, 0x12345678
mov al, 0xFF       ; EAX = 0x123456FF
mov ah, 0xAA       ; EAX = 0x1234AAFF
mov ax, 0xBBBB     ; EAX = 0x1234BBBB
```

> [!tip] 关键要点
> - EAX 是"累加器"，很多指令隐式使用它，返回值也放这里
> - ECX 是"计数器"，`LOOP` 和 `REP` 前缀自动使用
> - ESI/EDI 是字符串操作的源/目标指针
> - EBP/ESP 用于栈帧管理，不要挪作他用
> - 高 8 位寄存器（AH/BH/CH/DH）只存在于前 4 个寄存器
