# 并查集

并查集（Union-Find / Disjoint Set Union）是高效处理**不相交集合**的合并与查询操作的数据结构。

## 核心操作

- **Find(x)**：找到 x 所属集合的代表元素（根）
- **Union(x, y)**：合并 x 和 y 所在的集合

## 完整实现

```python
class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n
        self.count = n  # 连通分量数
    
    def find(self, x):
        """路径压缩"""
        while self.parent[x] != x:
            self.parent[x] = self.parent[self.parent[x]]  # 隔代压缩
            x = self.parent[x]
        return x
    
    def union(self, x, y):
        """按秩合并"""
        rx, ry = self.find(x), self.find(y)
        if rx == ry:
            return False
        if self.rank[rx] < self.rank[ry]:
            rx, ry = ry, rx
        self.parent[ry] = rx
        if self.rank[rx] == self.rank[ry]:
            self.rank[rx] += 1
        self.count -= 1
        return True
    
    def connected(self, x, y):
        return self.find(x) == self.find(y)
```

## 路径压缩详解

```
find(x):
    if parent[x] != x:
        parent[x] = find(parent[x])  # 递归：直接挂到根
    return parent[x]
```

递归版路径压缩更彻底（直接挂根），但可能栈溢出。迭代版（隔代压缩）更安全。

## 应用

### 连通分量计数

```python
def count_components(n, edges):
    uf = UnionFind(n)
    for u, v in edges:
        uf.union(u, v)
    return uf.count
```

### 等式判断（LeetCode 990）

```python
def equations_possible(equations):
    uf = UnionFind(26)
    for eq in equations:
        if eq[1] == '=':
            uf.union(ord(eq[0])-97, ord(eq[3])-97)
    for eq in equations:
        if eq[1] == '!':
            if uf.connected(ord(eq[0])-97, ord(eq[3])-97):
                return False
    return True
```

> [!tip] 复杂度分析
> - **单次操作**：近似 O(α(n))，α 为反阿克曼函数（增长极慢，≤5）
> - **m 次操作**：O(m · α(n)) ≈ O(m)
> - **空间**：O(n)

> [!info] 优化手段
> 1. **路径压缩**：Find 时将节点直接连到根
> 2. **按秩合并**：Union 时将矮树挂到高树下
> 3. 同时使用两种优化达到 O(α(n)) 摊还时间
