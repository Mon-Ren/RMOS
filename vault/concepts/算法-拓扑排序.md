# 拓扑排序

拓扑排序是对有向无环图（DAG）的顶点进行线性排序，使得对于每条有向边 (u, v)，u 在排序中出现在 v 之前。

## Kahn 算法（BFS + 入度）

维护入度表，每次将入度为 0 的顶点入队，取出后将其邻接点入度减 1。

```python
from collections import deque

def kahn_topo_sort(V, adj):
    in_degree = [0] * V
    for u in range(V):
        for v in adj[u]:
            in_degree[v] += 1
    
    queue = deque()
    for i in range(V):
        if in_degree[i] == 0:
            queue.append(i)
    
    order = []
    while queue:
        u = queue.popleft()
        order.append(u)
        for v in adj[u]:
            in_degree[v] -= 1
            if in_degree[v] == 0:
                queue.append(v)
    
    if len(order) != V:
        return None  # 存在环，非 DAG
    return order
```

## DFS 后序逆序

```python
def dfs_topo_sort(V, adj):
    visited = [False] * V
    order = []
    
    def dfs(u):
        visited[u] = True
        for v in adj[u]:
            if not visited[v]:
                dfs(v)
        order.append(u)  # 后序加入
    
    for i in range(V):
        if not visited[i]:
            dfs(i)
    
    return order[::-1]  # 逆序即拓扑序
```

> [!tip] 复杂度分析
> - **时间**：O(V + E) — 两种方法均是
> - **空间**：O(V) — 存储入度表 / 递归栈

> - 课程选修的先后顺序（LeetCode 207/210）
> - 任务调度、构建系统（Makefile）
> - 有向图判环
