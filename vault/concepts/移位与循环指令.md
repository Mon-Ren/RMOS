移位和循环指令将操作数的位向左或右移动，是位操作、乘除优化和加密算法的基本工具。

## 逻辑移位

### SHL — 逻辑左移 (Shift Logical Left)

```
  CF ← [b7 b6 b5 b4 b3 b2 b1 b0] ← 0
```

```asm
shl eax, 1         ; EAX 左移 1 位 = EAX * 2
shl eax, 3         ; EAX 左移 3 位 = EAX * 8
shl eax, cl        ; EAX 左移 CL 位（CL 是唯一合法的变量移位数）

; SHL 和 SAL 完全等价
```

### SHR — 逻辑右移 (Shift Logical Right)

```
  0 → [b7 b6 b5 b4 b3 b2 b1 b0] → CF
```

```asm
shr eax, 1         ; EAX 右移 1 位 = EAX / 2（无符号）
shr eax, 4         ; EAX 右移 4 位 = EAX / 16（无符号）
shr eax, cl        ; EAX 右移 CL 位
```

## 算术移位

### SAR — 算术右移 (Shift Arithmetic Right)

```
  b7 → [b7 b6 b5 b4 b3 b2 b1 b0] → CF
  （符号位复制，保留正负号）
```

```asm
mov eax, -16       ; 0xFFFFFFF0
sar eax, 1         ; EAX = -8  (0xFFFFFFF8)  → 保留符号
; vs
mov eax, -16
shr eax, 1         ; EAX = 0x7FFFFFF8  → 正数（丢失符号）

; SAR 用于有符号除法
mov eax, -100
sar eax, 2         ; EAX = -25（有符号除以 4）
```

### SAL — 算术左移

```asm
sal eax, 3         ; 等价于 SHL eax, 3（左移不需要区分逻辑/算术）
```

## 循环移位

### ROL — 循环左移 (Rotate Left)

```
  ┌──────────────────────────────┐
  ↓                              │
  [b7 b6 b5 b4 b3 b2 b1 b0] → CF
```

```asm
mov al, 0b10000001
rol al, 1          ; AL = 0b00000011, CF = 1（最高位转到最低位和 CF）
```

### ROR — 循环右移 (Rotate Right)

```asm
mov al, 0b10000001
ror al, 1          ; AL = 0b11000000, CF = 1（最低位转到最高位和 CF）
```

### RCL — 带进位循环左移 (Rotate through Carry Left)

```
  ┌─────────────────────────────────┐
  ↓                                 │
  CF → [b7 b6 b5 b4 b3 b2 b1 b0] → CF
  （CF 也参与旋转）
```

```asm
stc                  ; CF = 1
mov al, 0b00000000
rcl al, 1            ; AL = 0b00000001, CF = 0（CF 的 1 转入最低位）
```

### RCR — 带进位循环右移 (Rotate through Carry Right)

```asm
clc                  ; CF = 0
mov al, 0b00000001
rcr al, 1            ; AL = 0b00000000, CF = 1（最低位转入 CF，CF=0 转入最高位）
```

## 双精度移位

### SHLD — 双精度左移 (Double Precision Shift Left)

将目标操作数左移，空出的位由源操作数的高位填充。

```asm
; SHLD dest, src, count
; dest 左移 count 位，从 src 的高位取位填入

mov eax, 0x0000FFFF
mov ebx, 0x12340000
shld eax, ebx, 16    ; EAX = 0xFFFF1234
                      ; EBX 不变

; 用途：处理跨寄存器的位字段
```

### SHRD — 双精度右移 (Double Precision Shift Right)

```asm
; SHRD dest, src, count
; dest 右移 count 位，从 src 的低位取位填入

mov eax, 0x12345678
mov ebx, 0xABCDEF00
shrd eax, ebx, 16    ; EAX = 0xEF001234
                      ; EBX 不变
```

## 实用示例

### 用移位做乘除法

```asm
; EAX * 10 = EAX * 8 + EAX * 2
mov ebx, eax
shl eax, 3           ; EAX * 8
shl ebx, 1           ; EBX = EAX * 2
add eax, ebx         ; EAX = EAX * 10

; EAX / 4（有符号）
sar eax, 2
```

### 提取位字段

```asm
; 提取 EAX 的 bit 8-15
shr eax, 8
and eax, 0xFF

; 提取 bit 4-7（4 位字段）
mov ebx, eax
shr ebx, 4
and ebx, 0x0F
```

### MD5/SHA 中的位旋转

```asm
; SHA-256 中的 Σ0 函数
; Σ0(x) = ROTR(2) ^ ROTR(13) ^ ROTR(22)
mov ebx, eax
ror ebx, 2
mov ecx, eax
ror ecx, 13
xor ebx, ecx
ror eax, 22
xor eax, ebx         ; EAX = Σ0(原 EAX)
```

> [!tip] 关键要点
> - SHL/SHR 是逻辑移位，用 0 填充；SAR 是算术右移，用符号位填充
> - SHL n 位 = 乘以 2^n；SHR/SAR n 位 = 除以 2^n
> - ROL/ROR 的位不会丢失（循环到另一端）
> - RCL/RCR 连 CF 一起旋转，用于大数移位
> - SHLD/SHRD 实现跨寄存器移位，避免多次操作
