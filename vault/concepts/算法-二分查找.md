# 二分查找

在**有序数组**中高效查找目标值，每次将搜索范围缩小一半，O(log n) 时间。

## 标准模板

```python
def binary_search(arr, target):
    lo, hi = 0, len(arr) - 1
    while lo <= hi:
        mid = lo + (hi - lo) // 2  # 防整数溢出
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            lo = mid + 1
        else:
            hi = mid - 1
    return -1  # 未找到
```

## 循环不变量

二分查找的核心是维护循环不变量：
- `arr[0..lo-1]` 都 < target
- `arr[hi+1..n-1]` 都 > target
- target 只可能在 `arr[lo..hi]`

> [!tip] 区间选择
> `[lo, hi]` 闭区间 → `while lo <= hi`, `hi = mid - 1`
> `[lo, hi)` 左闭右开 → `while lo < hi`, `hi = mid`

## 左边界 / 右边界

```python
# 左边界：第一个 >= target 的位置
def lower_bound(arr, target):
    lo, hi = 0, len(arr)
    while lo < hi:
        mid = lo + (hi - lo) // 2
        if arr[mid] < target:
            lo = mid + 1
        else:
            hi = mid
    return lo

# 右边界：第一个 > target 的位置
def upper_bound(arr, target):
    lo, hi = 0, len(arr)
    while lo < hi:
        mid = lo + (hi - lo) // 2
        if arr[mid] <= target:
            lo = mid + 1
        else:
            hi = mid
    return lo
```

## 整数溢出防护

```python
# ❌ 危险：当 lo + hi > INT_MAX 时溢出
mid = (lo + hi) // 2

# ✅ 安全：始终正确
mid = lo + (hi - lo) // 2
```

## 应用场景

### 1. 旋转排序数组搜索

```python
def search_rotated(nums, target):
    lo, hi = 0, len(nums) - 1
    while lo <= hi:
        mid = lo + (hi - lo) // 2
        if nums[mid] == target:
            return mid
        # 判断哪半边有序
        if nums[lo] <= nums[mid]:  # 左半有序
            if nums[lo] <= target < nums[mid]:
                hi = mid - 1
            else:
                lo = mid + 1
        else:                       # 右半有序
            if nums[mid] < target <= nums[hi]:
                lo = mid + 1
            else:
                hi = mid - 1
    return -1
```

### 2. 搜索矩阵

```python
def search_matrix(matrix, target):
    """每行递增，下一行首元素 > 上一行末元素"""
    m, n = len(matrix), len(matrix[0])
    lo, hi = 0, m * n - 1
    while lo <= hi:
        mid = lo + (hi - lo) // 2
        val = matrix[mid // n][mid % n]  # 二维映射
        if val == target:
            return True
        elif val < target:
            lo = mid + 1
        else:
            hi = mid - 1
    return False
```

## 复杂度

- **时间**：O(log n)
- **空间**：O(1)（迭代版本）

## 关键要点

> [!important] 前提：有序
> 二分查找要求数组有序。无序数组需先排序 O(n log n)

> [!important] 二分的万能模板
> 很多"最大化最小值""最小化最大值"问题本质都是二分答案

> [!important] 闭区间 vs 开区间
> 统一用一种区间写法，避免 off-by-one 错误
