条件跳转指令根据 [[标志寄存器 EFLAGS]] 的状态决定是否跳转，是实现 if-else、循环等控制流的汇编基础。

## 条件跳转总表

### 无符号比较（基于 CF, ZF）

| 指令 | 含义 | 条件 |
|------|------|------|
| **JE / JZ** | 等于 / 为零 | ZF = 1 |
| **JNE / JNZ** | 不等于 / 非零 | ZF = 0 |
| **JA / JNBE** | 高于 (>) | CF = 0 ∧ ZF = 0 |
| **JAE / JNB / JNC** | 高于或等于 (≥) | CF = 0 |
| **JB / JNAE / JC** | 低于 (<) | CF = 1 |
| **JBE / JNA** | 低于或等于 (≤) | CF = 1 ∨ ZF = 1 |

### 有符号比较（基于 SF, OF, ZF）

| 指令 | 含义 | 条件 |
|------|------|------|
| **JE / JZ** | 等于 | ZF = 1 |
| **JNE / JNZ** | 不等于 | ZF = 0 |
| **JG / JNLE** | 大于 (>) | ZF = 0 ∧ SF = OF |
| **JGE / JNL** | 大于或等于 (≥) | SF = OF |
| **JL / JNGE** | 小于 (<) | SF ≠ OF |
| **JLE / JNG** | 小于或等于 (≤) | ZF = 1 ∨ SF ≠ OF |

### 单标志检查

| 指令 | 含义 | 条件 |
|------|------|------|
| **JC** | 进位 | CF = 1 |
| **JNC** | 无进位 | CF = 0 |
| **JO** | 溢出 | OF = 1 |
| **JNO** | 无溢出 | OF = 0 |
| **JS** | 符号（负数） | SF = 1 |
| **JNS** | 无符号（非负） | SF = 0 |
| **JP / JPE** | 偶校验 | PF = 1 |
| **JNP / JPO** | 奇校验 | PF = 0 |

## 有符号 vs 无符号：JA vs JG

```asm
; 比较 0xFFFFFFFF 和 0x00000001
mov eax, 0xFFFFFFFF
mov ebx, 0x00000001
cmp eax, ebx

; 无符号：0xFFFFFFFF(4294967295) > 0x00000001(1)
ja  unsigned_greater   ; ✓ 跳转（CF=0, ZF=0）

; 有符号：0xFFFFFFFF(-1) < 0x00000001(1)
jg  signed_greater     ; ✗ 不跳转（SF≠OF）
jl  signed_less        ; ✓ 跳转
```

> [!warning] JA vs JG 的选择
> - 无符号数比较用 **JA/JAE/JB/JBE**（Above/Below）
> - 有符号数比较用 **JG/JGE/JL/JLE**（Greater/Less）
> - 选错会导致 bug，比如把 -1 当作很大的正数

## 跳转范围

### 短跳转 (Short Jump)

- 偏移量：-128 到 +127 字节
- 编码：2 字节（Opcode + 8 位偏移）
- 汇编器自动选择最小编码

```asm
jmp short label      ; 显式短跳转
jmp label            ; 汇编器自动决定
```

### 近跳转 (Near Jump)

- 32 位偏移：±2GB
- 编码：5 字节（Opcode + 32 位偏移）

```asm
jmp near label       ; 显式近跳转
```

## 常见模式

### if-else 结构

```c
// C 代码
if (eax > ebx) {
    ecx = 1;
} else {
    ecx = 0;
}
```

```asm
; 汇编（有符号比较）
cmp eax, ebx
jle else_branch
    mov ecx, 1
    jmp end_if
else_branch:
    mov ecx, 0
end_if:

; 更优：使用 SETG（无分支）
cmp eax, ebx
setg cl              ; CL = (eax > ebx) ? 1 : 0
movzx ecx, cl
```

### while 循环

```c
// while (eax > 0) { eax--; }
```

```asm
loop_start:
    cmp eax, 0
    jle loop_end
    dec eax
    jmp loop_start
loop_end:
```

### 链式比较

```asm
; 检查值是否在 [10, 20] 范围内
cmp eax, 10
jl  out_of_range
cmp eax, 20
jg  out_of_range
    ; eax 在 [10, 20] 范围内
out_of_range:
```

## 编译器优化：消除分支

```asm
; 原始代码：有分支
cmp eax, 0
jne nonzero
    mov ebx, 0
    jmp done
nonzero:
    mov ebx, 1
done:

; 优化后：用 SETcc 替代
test eax, eax
setnz bl             ; BL = (eax != 0) ? 1 : 0
movzx ebx, bl
```

> [!tip] 关键要点
> - JA/JB 用于**无符号**比较，JG/JL 用于**有符号**比较
> - JE/JNE/JZ/JNZ 不区分有无符号（ZF 通用）
> - 短跳转 ±128B，近跳转 ±2GB
> - 现代 CPU 分支预测很重要，可考虑用 SETcc/CMOVcc 消除分支
> - 组合条件跳转：先 CMP，再连续条件跳转实现复杂逻辑
