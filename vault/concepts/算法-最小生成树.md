# 最小生成树 Prim/Kruskal

最小生成树（MST）是在无向连通带权图中选取 V-1 条边连接所有顶点，且总权值最小的树。

## Prim 算法

贪心扩展顶点：维护已选顶点集合 S，每次从 S 到 V\S 选权值最小的边对应的顶点加入。

```python
import heapq

def prim(adj, V):
    visited = [False] * V
    min_heap = [(0, 0)]  # (weight, vertex)
    total_weight = 0
    mst_edges = []
    
    while min_heap:
        w, u = heapq.heappop(min_heap)
        if visited[u]:
            continue
        visited[u] = True
        total_weight += w
        for v, weight in adj[u]:
            if not visited[v]:
                heapq.heappush(min_heap, (weight, v))
    
    return total_weight
```

**复杂度**：
- 邻接矩阵 + 暴力查找：O(V²)
- 邻接表 + 优先队列：O(E log V)

## Kruskal 算法

按边权排序，依次取最小边，用并查集判断是否成环。

```python
def kruskal(V, edges):
    edges.sort(key=lambda x: x[2])  # 按权值排序
    parent = list(range(V))
    
    def find(x):
        while parent[x] != x:
            parent[x] = parent[parent[x]]  # 路径压缩
            x = parent[x]
        return x
    
    def union(x, y):
        px, py = find(x), find(y)
        if px != py:
            parent[px] = py
            return True
        return False
    
    total = 0
    edge_count = 0
    for u, v, w in edges:
        if union(u, v):
            total += w
            edge_count += 1
            if edge_count == V - 1:
                break
    
    return total if edge_count == V - 1 else -1
```

**复杂度**：O(E log E) — 排序主导

## MST 唯一性

> [!warning] 唯一性条件
> MST 唯一当且仅当 **所有边权互不相同**。若有相同权值的边，可能存在多个不同的 MST（但权值总和相同）。

## 选择策略

> [!tip] Prim vs Kruskal
> - **稠密图**（E ≈ V²）：Prim + 邻接矩阵，O(V²)
> - **稀疏图**（E ≈ V）：Kruskal + 并查集，O(E log E)
> - 实际竞赛中 Kruskal 更常用（代码短，配合并查集效率高）
